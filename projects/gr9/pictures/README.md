This folder is for adding your images for the Grade 9 slideshow.

How to add images:
- If you're working locally, copy image files into this folder.
- If you're using Git, add, commit and push the files:

  git add projects/gr9/pictures/your-image.jpg
  git commit -m "Add picture for gr9 slideshow"
  git push

Notes:
- This static site can't save uploaded files from a browser without a server. Use the steps above to add files to the repository.
- You can preview images locally by opening `upload_preview.html` in the same folder (`projects/gr9`).
